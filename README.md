# Бот автоматизации Discord

### Автор не несет отвественности за ваши аккаунты, автоматизация аккаунтов пользователей (self-bots) запрещена правилами Discord

Этот бот позволяет входить на сервер по инвайт ссылке, писать сообщения в чате и оставлять реакцию на сообщение

Бот очень простой, без заморочек, может быть в будущем улучшу, по вашим пожеланиям

### Установка

Для запуска этого кода требуется Node.js версии 12 или выше. После установки Node.js, выполните следующие шаги:

### Склонируйте репозиторий:

```shell script
git clone https://github.com/0xZonaV/discord-auto.git
```
Установите зависимости:
```shell
cd discord-auto
```
```shell
npm install
```

### Измените файл tokens.txt в корневой директории проекта и добавьте в него свои токены и прокси в формате указаннй в tokens.txt.**
### Не забудьте удалить пример!!

### Запустите код, выполнив команду:
```shell
node index.js
```

Это запустит интерфейс командной строки, в котором вы можете выбрать нужную функцию для работы с кошельками.


### Как работать с ботом
1) Добавьте все ваши аккаунты и прокси в файл tokens.txt
2) В файле index.js в строке 20 укажите ваш API ключ от [2captcha](https://2captcha.com/?from=17205108)
3) Запустите бота
4) Выберите нужный пункт меню
5) Следуйте инструкциям
